This is a collection of programs for paper "Force, metric, or mass: Disambiguating causes of uniform gravity". The program uses the natural unit where the speed of light c=1.

(************************************)
Program.nb contains definitions of core functions. Run this notebook first before running other notebooks. 

Tests.nb is used to test subroutines, check maths, and plot diagnostic figures. Additional utility functions are defined in this notebook.

Examples.nb is used to generate figures in the paper. This notebook contains additional plotting functions.
